Community 5v5 Tool EUW · Custom Games
Not affiliated with Riot Games

Private 5v5 custom games, automated.

A community web application to create League of Legends 5v5 custom matches using Riot’s Tournament API (tournament codes), receive end-of-game callbacks, and compute internal performance metrics used for team balancing.

Repository GitHub Verification file /riot.txt Live app Vercel

What it does

  • 1
    Create matches

    Create private 5v5 custom games within a community (EUW).

  • 2
    Generate tournament codes

    Use Riot’s Tournament API to generate lobby codes for custom matches.

  • 3
    Post-game analytics

    Receive end-of-game callbacks, fetch match details, and compute internal “costs” metrics for balancing.

  • 4
    Discord integration

    Link players to Discord accounts and publish match summaries via a companion bot.

No matchmaking. No monetization. No prize pools. Metrics are internal and have no monetary value.

Technical summary

Platform: EUW
Mode: 5v5 blind pick (custom games)
APIs: Tournament API (codes + callbacks), Match API (post-game details)
Security: API key stored server-side only (never exposed to clients)
Data: minimal match-related data retained for community analytics
