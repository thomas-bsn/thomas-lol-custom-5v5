Community 5v5 Custom Games Tool

This project is a private community web application to organize League of Legends 5v5 custom games using Riot's Tournament API.

Features

  • Creation of private 5v5 custom matches
  • Generation of tournament codes
  • Retrieval of end-of-game data
  • Internal performance metrics for team balancing
  • Discord bot integration for match summaries

This project is not affiliated with Riot Games and is not monetized.
